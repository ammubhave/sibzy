*******************
Dynamic CSS Classes
*******************

..  glossary::
    
    .fb-login
        The element is visible only when the user is logged in.
        
    .fb-nologin
        The element is visible only when is not logged in.
        
    .fb-name
        The innerText is set to the user's full name
    
    .fb-username
        The innerText is set to the user's username
